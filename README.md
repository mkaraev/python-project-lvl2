# Gendiff
Command line tool for generating difference between two configuration files. 
Supported extentions:

 ```.json```
 
 ```.yml```

## Install
```pip install -i https://test.pypi.org/simple/ gendiff```
## Usage
``` gendiff /path/to/file1 /path/to/file2```
